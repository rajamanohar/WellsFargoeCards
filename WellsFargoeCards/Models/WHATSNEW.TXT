DESCRIPTION:
============
 This library has been developed to enable developers to quickly build applications
 that support VoiceVault's cloud-based voice biometrics for iOS (ViGo). It provides
 a simple interface that handles all the web service communication, voice verification
 process flow, and voice capture in the correct audio formats.
